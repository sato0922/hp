問4-(1) 年齢問題（一次方程式）
現在、子ども10歳・父44歳。
父の年齢が子どもの年齢の3倍になるのは何年後か。
1
文字を置く
x年後に3倍になるとする
2
x年後の年齢を式で表す
子ども：10 + x 歳 父　　：44 + x 歳
3
条件から方程式を立てる ← これが(i)の答え
44 + x = 3(10 + x)
4
解く
44 + x = 30 + 3x 14 = 2x x = 7
答え
(i) 44+x = 3(10+x)　　(ii) 7年後
✔ 確認：7年後 → 子17歳、父51歳。51÷17=3倍 ✓
🔄 同じ解き方・別パターン
現在、母32歳・子8歳。
母の年齢が子どもの年齢の3倍になるのは何年後か。
32 + x = 3(8 + x) 32 + x = 24 + 3x 8 = 2x x = 4
✔ 4年後 → 母36歳、子12歳。36÷12=3倍 ✓
問4-(2) 重さ問題（一次方程式）
物体A・B・Cがある。
・B = Aの3倍より100g重い
・C = 400g
A + C = B のとき、Aは何g？
1
文字を置く
物体Aの重さを x g とする
2
各物体を式で表す
A = x g B = 3x + 100 g C = 400 g
3
方程式を立てる
x + 400 = 3x + 100
4
解く
300 = 2x x = 150
答え
物体A = 150 g
✔ 確認：A=150, B=550, C=400。150+400=550=B ✓
🔄 同じ解き方・別パターン
物体A・B・Cがある。
・B = Aの2倍より30g重い
・C = 200g
A + C = B のとき、Aは何g？
x + 200 = 2x + 30 170 = x A = 170 g
✔ A=170, B=370, C=200。170+200=370=B ✓
問5 比例・反比例・グラフ選択

(1) 比例

yはxに比例し、x=−2のときy=10。x=3のときyを求めよ。
1
比例定数kを求める
y = kx に x=−2, y=10 を代入 10 = k×(−2) k = −5
2
x=3を代入
y = −5×3 = −15

(2) 反比例

反比例 y=a/x のグラフが点(−2,−8)を通るとき、aの値を求めよ。
1
座標を代入
a = x × y = (−2)×(−8) = 16 ※ 反比例 y=a/x は「a = xy」で求める

(3) グラフ選択

(i) y = −(3/2)x　　(ii) y = 3/x
それぞれのグラフとして適するものを1〜5から選べ。
1
グラフの特徴を確認
(i) y=−(3/2)x … 比例、原点通り、負の傾き → 右下がりの直線 → グラフ1 (ii) y=3/x … 反比例、a=3(正) → 第1・3象限の双曲線 → グラフ4
💡 グラフ選択のポイント：
・比例(y=kx) → 原点を通る直線。k>0なら右上がり、k<0なら右下がり
・反比例(y=a/x) → 双曲線。a>0なら第1・3象限、a<0なら第2・4象限
🔄 同じ解き方・別パターン
(1) yはxに比例し、x=4のときy=−12。x=−3のときyを求めよ。
(2) y=a/xのグラフが点(3,−6)を通るとき、aを求めよ。
(1) k = −12÷4 = −3 y = −3×(−3) = 9 (2) a = 3×(−6) = −18
問6 反比例・座標・回転体
曲線①：y=a/x　直線②：y=2x
点A：①と②の交点でx座標=3
点B：①上でx座標=9
点C：①と直線OBの交点（B以外）
点D：y軸上でy座標=8

(1) aの値

1
Aのy座標を求める
A のx=3 を y=2x に代入：y=6 よって A(3, 6)
2
y=a/xにAを代入
a = 3 × 6 = 18

(2) 点Cの座標

1
Bの座標を求める
x=9 を y=18/x に代入：y=2 B(9, 2)
2
直線OBの式を求める
傾き = 2÷9 = 2/9 直線OB：y = (2/9)x
3
①と直線OBの交点を求める
18/x = (2/9)x x² = 81 x = ±9 B はx=9 なので C は x=−9 y = 18÷(−9) = −2 C = (−9, −2)

(3) 三角形DOAをy軸で回転した立体の体積

1
各点の座標を整理
O(0,0)、D(0,8)、A(3,6)
2
y=0〜6の部分（辺OA = y=2x → x=y/2）
円錐：底面半径3、高さ6 V₁ = (1/3)×π×3²×6 = 18π
3
y=6〜8の部分（辺DA）
DAの式：x = (8−y)×3/2 y=6でx=3、y=8でx=0 → 逆円錐 V₂ = (1/3)×π×3²×2 = 6π
4
合計
V = 18π + 6π = 24π cm³
答え
(1) a=18　(2) C(−9,−2)　(3) 24π cm³
